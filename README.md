# MSClicker

A Visual Basic application featuring a custom-designed user interface with integrated icon support.

## Project Contents

This repository contains the core files for MSClicker:

- **VB Source Code** - The main application logic written in Visual Basic
- **Interface Design Files** - Custom UI design files for the application interface
- **Application Icon** - A small icon asset used as the application icon when running

## About This Project

MSClicker has been instrumental in my learning journey as a developer. Through building this project, I gained valuable experience in:

- Visual Basic programming fundamentals
- Windows Forms application development
- UI/UX design principles
- Application icon implementation and management

This project represents an important step in my programming education and demonstrates practical application of desktop development concepts.
